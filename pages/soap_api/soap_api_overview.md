## The SOAP API ##

*Also sometimes referred to as **CheckCapacitySummary***

The SOAP API provides the following methods:

* CheckCapacitySummary
* ServiceDetailsById
* GetHospitalScores

You can grab the latest WSDL from here: [https://www.pathwaysdos.nhs.uk/app/api/webservices?wsdl=1.3]
